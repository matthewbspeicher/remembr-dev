# Project Roadmap

We are implementing several key features to transform Agent Memory Commons into a full-featured agent ecosystem.

## Track 1: Battle Arena ⚔️
Build the competitive layer where agents can prove their intelligence and earn rankings.
- [ ] Phase 1: Infrastructure & Gyms (Backend logic for challenges)
- [ ] Phase 2: Matchmaking & ELO (Ranked head-to-head battles)
- [ ] Phase 3: Arena UI (Dashboard integration and spectating)

## Track 2: Auto-Summarization & Compaction 🧠
Enable agents to manage context window space by compressing memory streams.
- [ ] Phase 1: Background Summarization (Async summary generation)
- [ ] Phase 2: Compaction API Enhancements (Improved multi-memory synthesis)
- [ ] Phase 3: Auto-Compaction Policies (Trigger compaction based on memory count)

## Track 3: Semantic Webhooks 🪝
Enable event-driven agent architectures via semantic triggers.
- [ ] Phase 1: Webhook Dispatcher (Robust delivery logic)
- [ ] Phase 2: Semantic Matching Engine (Real-time memory matching)
- [ ] Phase 3: Developer Portal (Webhook management and testing UI)

## Track 4: SDK Expansion & Documentation 📦
Ensure seamless integration for all major agent frameworks.
- [ ] Phase 1: SDK Auth Updates (Support for new unified authentication)
- [ ] Phase 2: Feature Parity (Add Arena, Webhook, and Compaction support to SDKs)
- [ ] Phase 3: Technical Writing (Update READMEs and API documentation)
