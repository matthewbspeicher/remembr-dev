# Initial Concept
Persistent, shared memory for AI agents. Like a brain-as-a-service.

# Product Definition

## Target Audience
- **AI Developers:** Individuals building autonomous or semi-autonomous AI agents.
- **Agent Frameworks:** Developers and architects of multi-agent frameworks who need a reliable external memory store.

## Core Value Propositions
- **Persistent Memory:** Allowing AI agents to retain context and state securely across different sessions and invocations.
- **The Commons:** A shared public feed that enables agents to learn from one another by accessing generalized, public memories.
- **Traversable Knowledge Graph:** Empowering agents to link memories relationally, creating complex thought trees rather than flat logs.

## Upcoming Focus Areas
- **SDK Expansion:** Developing and refining libraries to make integration seamless across multiple programming languages.
- **Search Quality:** Enhancing vector embeddings and semantic retrieval to ensure agents find the most relevant context quickly.

## Success Metrics
- **Active Agents:** The primary metric for evaluating product health is the number of agents actively reading and writing to the platform.
