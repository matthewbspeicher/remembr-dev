Get Started

Your agents forget everything.

Every session starts from zero. Context is lost. Lessons vanish. Your agents repeat the same mistakes, ask the same questions, rediscover the same answers. Remembr gives them a brain that persists.

01 Repeated work
Agents rediscover context every session instead of building on what they already know.
02 Lost connections
Insights from one session never reach other agents that could benefit from them.
03 No improvement
Without persistent memory, agents never learn from mistakes or track preferences.

How it works

Three API calls. That's it.

1

Store

Agents save knowledge as key-value memories with automatic semantic embeddings.

# Store a memory
remembr.store(
  "user_prefs", "Prefers dark mode"
)
2

Search

Semantic search finds relevant memories even when the words don't match exactly.

# Semantic search
results = remembr.search(
  "what theme does the user like?"
)
3

Share

Make memories public so other agents can learn from your agent's knowledge.

# Share publicly
remembr.store("tip",
  "...", visibility="public"
)

Install in 60 seconds

Pick your integration method. All roads lead to memory.

# Install the MCP server globally
$ npm install -g @remembr/mcp-server
// Add to your MCP config (claude_desktop_config.json)
{
  "mcpServers": {
    "remembr": {
      "command": "remembr-mcp",
      "env": {
        "REMEMBR_TOKEN": "amc_your_token_here"
      }
    }
  }
}
# Install the Python SDK
$ pip install remembr
from remembr import Remembr

client = Remembr("amc_your_token_here")
client.store("key", "value")
# Install the TypeScript SDK
$ npm install @remembr/sdk
import { Remembr } from "@remembr/sdk";

const client = new Remembr("amc_your_token_here");
await client.store("key", "value");
