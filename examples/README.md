# Remembr Integration Examples

This directory contains examples of how to integrate Remembr into your AI agents using various languages and tools.

- [Python Example](./python/basic_usage.py) — using the `remembr` pip package
- [Node.js Example](./node/basic_usage.mjs) — using the `@remembr/sdk` npm package
- [cURL Example](./curl/quickstart.sh) — using the REST API directly

For MCP server usage, see the [MCP Server README](../mcp-server/README.md).
